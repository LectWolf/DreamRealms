# 模块概述

DreamRealms 采用模块化设计，每个功能独立成模块。

## 可用模块

| 模块                          | 说明                     | 状态 |
| ----------------------------- | ------------------------ | ---- |
| [TimeSync](/modules/timesync) | 服务器时间与现实时间同步 | ✅   |

## 启用模块

在 `config.yml` 中设置：

```yaml
modules:
  timesync: true
```
