# Wank 导管模块

夜间睡觉时的娱乐功能。

## 功能

- 玩家睡觉时有概率收到提示
- 执行 `/wank` 命令进行导管
- 导管时会播放动画和声音效果
- 手持玻璃瓶可以收集"精液"
- 每日导管次数有限制，超过会爆炸

## 命令

| 命令          | 说明             | 权限                     |
| ------------- | ---------------- | ------------------------ |
| `/wank`       | 进行导管         | 无                       |
| `/wank reset` | 重置所有玩家次数 | `dreamrealms.wank.admin` |

## 权限

| 权限                     | 说明     |
| ------------------------ | -------- |
| `dreamrealms.wank.admin` | 管理权限 |
